** 2.2 Layer 3 backend interface

Layer 3 = display backend、本 repo は *TUI reference impl のみ提供*。
GTK / Cocoa / Win32 / wgpu は community が独立 repo で実装、本 repo の
=emacs-tui-backend.el= を precedent として参照。

backend interface (= NeLisp Doc 33 v2 §6 display-spec contract):

| API category | contract | TUI 実装 | future GUI 実装 |
|--------------+----------+----------+-----------------|
| frame / window / canvas | =nelisp-display-frame-create= / =nelisp-display-window-create= 等 | ncurses window | gtk_window_new / NSWindow / CreateWindow |
| drawing operations | =nelisp-display-text-draw= / =nelisp-display-line-draw= 等 | ncurses mvprintw | cairo / Quartz / GDI |
| event sources | =nelisp-display-event-poll= / =nelisp-display-key-read= 等 | getch + select | gtk event / NSEvent / WindowProc |
| layout primitive | =nelisp-display-layout-vbox= / =nelisp-display-layout-hbox= | row/col 計算 | gtk_box / NSStackView / Win32 layout |
| font / face / image | =nelisp-display-face-set= / =nelisp-display-image-load= 等 | terminfo color / no image | freetype / CoreText / DirectWrite |
| input method | =nelisp-display-im-context-create= 等 | terminal IME 経由 | IBus / IMK / IMM32 |

*reference impl の scope* (詳細 Phase 2 §3.2):
- ncurses-style TUI で =text + basic-color + keyboard + resize= の minimum
  set のみ (= NeLisp Doc 33 v2 §4.4 の v2 改訂 + Doc 43 §3.1 Phase 11.A
  ship gate と整合、*image / path / IME は v2.x 以降後送り*)

  TUI MVP 列 = =text= / =basic-color= / =keyboard= / =resize= / =layout-box=
  / =layout-grid=) を declare
- *capability 未対応時 fallback = Doc 43 v1 §2.5a degrade contract LOCKED
  rule に準拠* (T130 MINOR M-3 反映): 未対応 capability への API 呼び出し
  は =display-spec-unsupported= condition signal、condition data plist =
  =:capability= / =:api= / =:backend= の 3 件、application 側 portable

  前 guard、stale capability cache 検出も同 condition で error 化。本 repo
  Phase 2 TUI backend = =image= / =mouse= / =ime-active= 等 unsupported
  declare 時 fallback (= 例 image → space + alt-text 表示) を Doc 43 §2.5a
  と整合する形で実装
- Phase 11 v2.0 ship gate と整合
- *Phase 11.B (= 本 repo Phase 3) 移送物* = Doc 43 §3.2 で decompose 済の
